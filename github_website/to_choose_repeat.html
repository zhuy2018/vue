Git 常用命令表

master 默认开发分支   Head 默认开发分支  Head^Head的父提交  origin 默认远程版本库

创建版本库
$ git clone git@git.oschina.net:veki/veki.git         克隆远程版本库
$ git init                                                                初始化本地git仓库（创建新仓库）

修改和提交
$ git status                                                           查看当前仓库状态
$ git add .                                                             跟踪所有改动的文件
$ git add <file>                                                      跟踪指定的文件                                                     
$ git mv <old> <new>                                            修改文件名字
$ git rm <file>                                                        删除文件                                                         
$ git rm --cached <file>                                         停止跟踪文件但不删除

查看历史提交
$ git log                                                                 查看提交历史
$ git log -p <file>                                                   查看指定文件的提交历史
$ git blam                                                              以列表方式查看指定文件的提交历史

撤销
$ git reset --hard HEAD                                         撤销工作目录中未提交文件的修改内容
$ git checkout HEAD <file>                                    撤销指定的未提交文件的修改内容
$ git rever <commit>                                              撤销指定的提交

分支与标签
$ git branch                                                           显示所有本地分支
$ git checkout <branch/tag>                                  切换到指定分支或标签
$ git checkout <new-tag>                                      创建新分支
$ git branch -d <branch>                                       删除本地分支
$ git tag                                                                 列出所有本地标签
$ git tag <tagname>                                              基于最新提交创建标签
$ git tag -d <tagname>                                          删除标签

合并
$ git marge <branch>                                            合并指定分支到当前分支
$git rabase <branch>                                            衍合指定分支到当前分支

远程操作
$ git remote -v                                                       查看远程版本库信息
$ git remote show <remote>                                  查看指定远程版本库信息
$ git remote add <remote> <url>                           添加远程版本库
$ git fetch <remote>                                              从远程库获取代码
$ git pull <remote> <branch>                                下载代码及快速合并
$ git push <remote> <branch>                              上传代码及快速合并
$ git push <remoto> : <branch/tag-name>            删除远程分支或标签
$ git push --tags                                                    上传所有标签
